## 🚀 Cara Install

### Step 1 — Install Termux
Download **Termux** dari [F-Droid](https://f-droid.org/packages/com.termux/) (bukan Play Store).
Jalankan ```bash
termux-change-repo
```

### Step 2 — Jalankan command install
Buka Termux, lalu ketik command berikut:

```bash
curl -sL https://raw.githubusercontent.com/Wayy1702/mytoolbox/main/install.sh | bash
```

Tunggu hingga proses selesai (download dependencies otomatis).

### Step 3 — Jalankan toolbox
Setelah install selesai, ketik:

```bash
wayy
```

### Step 4 — Masukkan License Key
Saat pertama kali dijalankan, akan muncul layar aktivasi:

```
╔══════════════════════════════════════╗
║   W A Y Y  T O O L B O X  v4.5      ║
║         AKTIVASI LISENSI             ║
╚══════════════════════════════════════╝

  HWID Device: 40bb9f70f93bc6a29547...

  Masukkan key pembelian kamu:
  (Ketik key lalu tekan Enter)
  Key →
```

Ketik key yang sudah kamu beli, lalu tekan **Enter**.

> ⚠️ **1 key hanya untuk 1 device.** Key yang sudah aktif di device A tidak bisa dipakai di device B.

## 🔑 Sistem Lisensi

| Status | Keterangan |
|--------|-----------|
| ✅ **AKTIF** | Key valid, device terdaftar |
| ❌ **BANNED** | Key di-ban oleh admin |
| ⚠️ **HWID MISMATCH** | Key sudah dipakai di device lain |
| 🔄 **UNUSED** | Key belum pernah dipakai |

**Aturan key:**
- 1 key = 1 device
- Key yang sudah aktif di device lain **tidak bisa** dipakai
- Jika ingin pindah device, hubungi admin untuk **Reset HWID**

## 📱 Cara Pakai

### Tambah Akun
```
Menu → 2 (Edit App List) → 1 (Tambah App)
```
Isi nama akun, package name Roblox, dan deeplink game.

### Auto Rejoin
```
Menu → 1 (Auto Rejoin)
```
Toolbox akan mengirim deeplink ke semua akun secara otomatis sesuai interval yang diset.

### Setting Discord Webhook
```
Menu → 4 (Settings) → Discord Webhook
```
Paste URL webhook Discord kamu untuk dapat notifikasi status.

## ❓ FAQ

**Q: Kenapa muncul "Tidak dapat terhubung ke server lisensi"?**  
A: Pastikan koneksi internet aktif. Kalau pakai data, coba ganti ke WiFi atau sebaliknya.

**Q: Key saya bilang "HWID MISMATCH" padahal saya sendiri yang pakai?**  
A: Kemungkinan kamu pernah install di device lain atau factory reset. Hubungi admin untuk reset HWID.

**Q: Toolbox tidak mau update?**  
A: Jalankan `wayy` — toolbox auto-update setiap dijalankan. Pastikan internet aktif.

**Q: Bagaimana cara uninstall?**  
A: Di menu toolbox → pilih **8 (Uninstall)**.
